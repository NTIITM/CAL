# Datasets

This directory holds per-dataset subdirectories, each containing:

- `<dataset>.h5ad` — single-cell RNA-seq expression data in AnnData format
- `geneformer_ds/` — pre-tokenised HuggingFace `datasets` shards for Geneformer
- `description.txt` — (optional) brief dataset metadata

**Data files are excluded from git** (see `.gitignore`) to keep the repository lightweight.

---

## Supported datasets

| Dataset | Cells | Cell types | Used by |
|---------|------:|:----------:|---------|
| `hPancreas` | ~14 k | 13 | scGPT, CellFM |
| `hBone` | ~11 k | 10 | scGPT, CellFM, Geneformer |
| `MS` | ~25 k | 7 | CellFM |

---

## Data availability

These datasets are derived from publicly available single-cell studies.
The processed `.h5ad` files used in MH-CAL experiments are available via **Google Drive** (link to be released upon paper acceptance).

To download with `gdown`:
```bash
pip install gdown

# Example for hPancreas
gdown "https://drive.google.com/uc?id=<FILE_ID>" -O data/hPancreas/hPancreas.h5ad
```

A one-click download script `scripts/download_data.sh` will be provided alongside the Google Drive link.

## Geneformer tokenisation

The `geneformer_ds/` subdirectory is generated by pre-tokenising the `.h5ad` file
with the Geneformer tokeniser. Pre-generated tokenised datasets will also be
available on the same Google Drive folder.

To re-generate manually:
```bash

python -c "
# See Geneformer docs for tokenization script
"
```
